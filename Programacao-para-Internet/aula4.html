Formulários

O HTML possui a tag form que possibilita definir formulários para submissão de dados informados pelo usuário por meio de campos input, dentro de um formulário. A tag <tr> define uma linha em uma tabela. exemplo: 

    <table>
      <tr> <td>Celula 1x1 </td> Celula 1x2 </td> 
      </tr>
      <tr>
      <td>celula 2x1</td> <td>celula 2x2</td>
      </tr>
    </table>
O codigo anterior define a tabela abaixo
Carnes ID Item churrasco quantidade Valor
1 Picanha 1kg R$ 59,90
2 Peito Frango 2kg R$ 29,90
Bebidas 3 Cachaça 3L R$ 39,90
4 Cerveja 12L R$ 49,90
