O que é JavaScript?

JavaScript foi criado para trazer interatividade às páginas web. Diferente de linguagens de servidor (como PHP ou Java), ele roda diretamente no navegador do cliente, permitindo respostas rápidas sem recarregar a página. O navegador cria um ambiente seguro chamado "sandbox" para impedir que o código acesse arquivos do computador do usuário indevidamente. Objetos globais importantes incluem o document (para manipular o HTML), console (para depuração) e alert (para mensagens).
